<h2>Week 1 – Activity 1: Simple Scene with a Moving Node</h2>

<p>
This activity demonstrates creating a basic scene in <b>Godot Engine</b> and implementing
a simple moving node using <b>GDScript</b>. The objective is to understand the basic
structure of scenes, nodes, and scripting in Godot.
</p>

<h3>Key Concepts</h3>
<ul>
<li>Scene creation in Godot</li>
<li>Node hierarchy</li>
<li>Basic scripting with GDScript</li>
<li>Object movement</li>
</ul>

<h2>Week 2 – Activity 2: Level Design</h2>

<p>
This activity focuses on designing levels using tilemaps and implementing
basic hazards for gameplay challenges.
</p>

<h3>Concepts Covered</h3>
<ul>
<li>Tilemaps for grid-based level creation</li>
<li>Adding hazards such as spikes or traps</li>
<li>Designing level pacing</li>
<li>Creating difficulty curves</li>
</ul>

<h3>Activity</h3>
<p>
Design two levels for an <b>Endless Runner Game</b> (2D or 3D).

<h2>Week 3 – Activity 1: UI/UX & Audio</h2>

<p>

through Godot's audio system.

<h3>Subtopics</h3>

<h2>Week 4 – Activity 1: 3D Basics & Optimization</h2>

<li>Basic 3D object placement</li>
<li>Performance optimization techniques</li>
</ul>
